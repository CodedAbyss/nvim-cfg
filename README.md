# nvim-cfg

### Git Clone Installation

`git clone https://github.com/codedabyss/nvim-cfg.git ~/.config/nvim --depth 1` (Linux / MacOS)

`git clone https://github.com/codedabyss/nvim-cfg.git ~/appdata/local/nvim --depth 1` (Windows Powershell)

`git clone https://github.com/codedabyss/nvim-cfg.git %localappdata%/nvim --depth 1` (Windows Cmd Prompt)
